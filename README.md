# 1. Clone the repository

```bash
git clone https://github.com/GunawardhanaSIA/Year_4_Research_Implementation.git
```
```bash
cd Research---Year-4
```

# 2. Install dependencies

```bash
npm install
```
   		
# 3. Start the game 
You have two options to run the game locally:
   
Option 1: Using Node.js server
```bash
npm start
```
 Option 2: Using Live Server
   
   i. Install Live Server locally
```bash
npm install --save-dev live-server
```
     					
   ii. Now run it
```bash
live-server --port=5500 
```
